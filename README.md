# DSA Case Studies – TourPlan Smart Tourism & Travel Itinerary Optimization System

## Overview

This repository contains **six case studies** developed for the **DSA-2 course** based on the domain **TourPlan – Smart Tourism & Travel Itinerary Optimization System**.

The project demonstrates the application of **Data Structures and Algorithms** in solving real-world tourism and travel management problems such as destination indexing, hotel budget analysis, city connectivity, package ranking, itinerary optimization, travel route recommendation, and smart tourism application development using advanced non-linear data structures.

---

## Case Studies Included

### CO1 – Tourist Destination Indexing using AVL Tree

- Implemented AVL Tree for storing tourist destinations.
- Performed insertion and search operations.
- Applied AVL rotations to maintain tree balance.
- Achieved O(log n) search efficiency.

### CO2 – Hotel Budget Range Query System using Segment Tree

- Implemented Segment Tree for hotel price management.
- Supported range minimum price queries.
- Enabled efficient hotel budget analysis.
- Achieved O(log n) query performance.

### CO3 – Tourist City Connectivity using BFS, DFS and Prim's Minimum Spanning Tree

- Modeled tourist cities as a weighted graph.
- Implemented BFS and DFS graph traversal techniques.
- Constructed a Minimum Spanning Tree using Prim's Algorithm.
- Optimized city connectivity and travel network cost.

### CO4 – Tourist Package Ranking and Travel Itinerary Optimization using Heap Sort, Greedy and Dynamic Programming

- Implemented Heap Sort for tourist package ranking.
- Applied Counting Sort and Radix Sort for package organization.
- Used Greedy Knapsack for budget-based package selection.
- Implemented Longest Common Subsequence (LCS) for tourist preference analysis.
- Applied Matrix Chain Multiplication (MCM) for itinerary optimization.

### CO5 – Tourist Route Recommendation and Travel Package Optimization using Non-Linear Data Structures, Greedy and Dynamic Programming Algorithms

- Represented tourist destinations using graph-based structures.
- Recommended efficient travel routes.
- Applied Greedy Algorithms for package selection.
- Used Dynamic Programming for itinerary planning.
- Optimized travel cost, distance, and tourist preferences.

### CO6 – Smart Tourism & Travel Itinerary Optimization System using Advanced Non-Linear Data Structures and Application Development

- Developed the TourPlan application using advanced non-linear data structures.
- Represented tourist destinations and city connectivity using Graphs.
- Used weighted edges based on travel distance, time, and cost.
- Organized destinations using Tree-based structures.
- Implemented route recommendation and shortest-path concepts.
- Managed hotels, travel packages, and destination information efficiently.
- Generated optimized travel itineraries based on user preferences and budget constraints.
- Demonstrated real-world application development using non-linear data structures.
