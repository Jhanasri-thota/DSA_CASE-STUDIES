# DSA Case Studies – TourPlan Smart Tourism & Travel Itinerary Optimization System

## Overview

This repository contains three case studies developed for the DSA-2 course based on the domain **TourPlan – Smart Tourism & Travel Itinerary Optimization System**. The case studies demonstrate the application of advanced data structures and algorithms in solving real-world tourism and travel management problems.

---

## Case Studies Included

### CO1 – Tourist Destination Indexing using AVL Tree
- Implemented AVL Tree for storing tourist destinations.
- Performed insertion and search operations.
- Applied AVL rotations to maintain tree balance.
- Achieved O(log n) search efficiency.

### CO2 – Hotel Budget Range Query System using Segment Tree
- Implemented Segment Tree for hotel price management.
- Supported range minimum price queries.
- Enabled efficient hotel budget analysis.
- Achieved O(log n) query performance.

### CO3 – Tourist City Connectivity using BFS, DFS and Prim's Algorithm
- Modeled tourist cities as a weighted graph.
- Implemented BFS and DFS graph traversal techniques.
- Constructed a Minimum Spanning Tree using Prim's Algorithm.
- Optimized city connectivity and travel network cost.
